# Credit Risk Analysis Dashboard

## 📊 Project Overview
This project analyzes customer credit data to predict default risk using machine learning and visualize insights using Tableau.

## 🔧 Tools Used
- Python (Pandas, Scikit-learn)
- Tableau
- Logistic Regression

## 📈 Key Features
- Predicts high-risk customers
- Risk segmentation (High vs Low)
- Interactive dashboard
